# TUTORIAL GUI

## Connect to database
To connect to a SQL database, you can do:

```bash
xelo2
```
and then log in with the prompt screen. Or you can pass the credentials directly (in plain text...):

```bash
xelo2 --mysql DATABASE_NAME -U USERNAME -P PASSWORD
```
